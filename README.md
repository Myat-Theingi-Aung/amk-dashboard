# AMK POS System Dashboard

## Requirements
- Node 14 or higher
- MySQL 5.7 or 8

## Installation

Clone the repo locally:
```
git clone https://github.com/Myat-Theingi-Aung/amk-dashboard.git
```

`cd` into cloned directory and install dependencies. run below command one by one.
```bash
npm install
```

## Server Run

Run the dev server:
```
npm start
```

Visit below url:
```
http://localhost:3000
```
